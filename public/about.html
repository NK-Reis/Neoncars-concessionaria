Sobre o projeto

Este é um projeto didático de uma concessionária fictícia criado com HTML, CSS, JavaScript no front-end. O acesso aos preços é liberado apenas para usuários logados (simulação no front-end). Em etapas futuras, será adicionado um back-end com Node.js + SQLite (SQL) para cadastro, login e regras de acesso reais.

Tecnologias

  • HTML5 + CSS3 (tema escuro roxo)
  • JavaScript (validação, estado de login e exibição condicional de preços)
  • SQL (planejado): SQLite para usuários e carros
  • Git + GitHub (versionamento e publicação)
